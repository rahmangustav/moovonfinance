# Kamus Istilah Keuangan — Moovon Finance

Istilah andalan + cara Moovon menjelaskannya (analogi santai, sekali jelasin per video saat pertama muncul).

| Istilah | Penjelasan gaya Moovon |
|---|---|
| Inflasi | Kenaikan harga barang secara umum — duit lo "menyusut" daya belinya. |
| Suku bunga BI (BI Rate) | "Harga sewa duit" yang ditetapkan Bank Indonesia; naik = kredit mahal, turun = kredit murah. |
| Obligasi / SBN | Surat utang — lo minjemin duit ke negara/perusahaan, dibayar bunga (kupon) rutin. |
| Dividen | Bagi hasil laba perusahaan ke pemegang saham. |
| Diversifikasi | Jangan taruh semua telur di satu keranjang. |
| Blue chip | Saham perusahaan besar, mapan, likuid — "pemain senior" di bursa. |
| Compound interest | Bunga-berbunga: bunga yang menghasilkan bunga lagi. Snowball effect. |
| IHSG | Indeks Harga Saham Gabungan — "rapor rata-rata" seluruh saham di BEI. |
| Likuiditas | Seberapa gampang aset dicairin jadi uang tunai tanpa banting harga. |
| PER (P/E ratio) | Berapa tahun laba perusahaan "balikin" harga sahamnya — makin kecil relatif makin murah. |
| Kupon | "Uang sewa" yang dibayar penerbit obligasi ke lo secara rutin — analogi duit lo "ngekos" di negara. |
| Tenor | Lamanya kontrak pinjaman — pas habis (jatuh tempo), pokok balik utuh. |
| Floating with floor | Kupon ngikutin BI-Rate tapi ada "lantai" batas bawah yang gak bisa ditembus. |
| Pasar sekunder | Tempat jual-beli surat berharga "bekas" antar investor, setelah masa penawaran perdana. |
| Early redemption | Fasilitas nyairin sebagian dana SBR/ST sebelum jatuh tempo — terbatas dan ada jendela waktunya. |
| SID | Single Investor Identification — "KTP investor" buat transaksi di pasar modal Indonesia. |

> Tambah istilah baru di sini tiap kali dipakai di video, biar penjelasan konsisten antar video.
