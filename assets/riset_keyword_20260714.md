# Riset Keyword Evergreen — 14 Jul 2026

Metode: (1) YouTube autocomplete `hl=id&gl=id` (permintaan pencarian nyata) untuk
12 seed → (2) audit kompetisi top-8 hasil pencarian per keyword via YouTube Data
API (views, umur video, subscriber channel pesaing). Sinyal peluang = permintaan
tinggi + video pesaing sudah tua (>2 th) + ada channel kecil yang ikut ranking
(bukti keyword bisa ditembus tanpa channel besar).

## Temuan pola dari autocomplete

- **Tutorial berbasis aplikasi** mendominasi: "di stockbit", "di ajaib", "di bibit"
  muncul di hampir semua seed. Permintaan raksasa (video top 1,3 JUTA views).
- **"dari nol"** adalah frasa magnet pemula ("belajar saham dari nol" mengalahkan
  seed-nya sendiri di autocomplete).
- **Dividen** punya ekor panjang: terbaik / tertinggi / bulanan / jangka panjang / jumbo.
- **Tag tahun** ("2026") dicari aktif → konten wajib segar, pesaing lama gugur sendiri.
